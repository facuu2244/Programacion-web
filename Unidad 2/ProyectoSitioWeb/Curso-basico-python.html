Objetivos

  • Aprenderás a instalar Python y configurar VSCode, preparándote para desarrollar programas desde cero.
  • Dominarás la sintaxis básica de Python, incluyendo variables, operadores, condicionales, y bucles para resolver problemas lógicos.
  • Serás capaz de trabajar con estructuras de datos como listas, diccionarios, conjuntos y tuplas, manipulando información de forma eficiente.
  • Podrás definir y aplicar funciones en tus proyectos, mejorando la organización y modularidad del código.
  • Comprenderás y pondrás en práctica los principios de la programación orientada a objetos (POO), como clases, objetos, herencia y polimorfismo.
  • Aprenderás a gestionar errores y excepciones en tus programas, aumentando la robustez y fiabilidad del código.
  • Serás capaz de utilizar bibliotecas estándar de Python para manipulación de archivos, manejo de fechas, y otras tareas comunes en el desarrollo.
  • Implementarás algoritmos de búsqueda y ordenación como búsqueda binaria, quicksort, mergesort, mejorando el rendimiento de tus programas.
  • Dominarás el uso de expresiones regulares para la búsqueda y manipulación avanzada de textos.
  • Podrás trabajar con archivos JSON, CSV, y XML para manejar y procesar grandes volúmenes de datos.
  • Serás capaz de automatizar tareas repetitivas utilizando scripts en Python, desde la administración de archivos hasta el web scraping.
  • Aprenderás a crear e interactuar con bases de datos SQLite y MySQL, realizando consultas SQL desde Python.
  • Comprenderás cómo realizar pruebas unitarias (unit testing) para garantizar la calidad y funcionalidad de tu código.
  • Serás capaz de trabajar con APIs REST y consumir servicios web desde tus programas en Python.
  • Desarrollarás proyectos prácticos, como un web scraper, una API REST, o una aplicación de línea de comandos.

Descripcion del curso

¿Listo para dar el salto y convertirte en un experto en Python? Nuestro curso está diseñado para llevarte desde los fundamentos de la programación hasta un nivel avanzado, dominando uno de los lenguajes más versátiles y demandados en el mundo de la tecnología.

Comenzarás aprendiendo a instalar Python y a configurar VSCode, tu entorno de desarrollo. Desde allí, te guiaremos a través de los conceptos esenciales de Python, incluyendo tipos de datos, operadores y estructuras de control como bucles y condicionales. Pero eso es solo el principio. A medida que avances, profundizarás en estructuras de datos como listas, diccionarios, tuplas y conjuntos, y te convertirás en un maestro de la programación orientada a objetos (POO), comprendiendo conceptos clave como clases, objetos, herencia y polimorfismo.

Además, aprenderás a aplicar algoritmos de búsqueda y ordenación para optimizar tus programas, y serás capaz de gestionar archivos de manera eficiente con Python. También nos adentraremos en el manejo de excepciones y pruebas unitarias para asegurar la calidad y robustez de tu código, dándote las habilidades necesarias para crear programas fiables.

En la parte avanzada del curso, explorarás cómo interactuar con bases de datos como SQLite y MySQL, así como el consumo de APIs REST para desarrollar aplicaciones modernas que interactúan con servicios web. No solo aprenderás teoría, sino que desarrollarás proyectos prácticos, como un web scraper, una API REST, y la automatización de tareas, que te permitirán poner en práctica todo lo aprendido.

Este curso no es solo una introducción a Python; es un recorrido completo desde cero hasta un nivel avanzado que te proporcionará las herramientas y el conocimiento necesarios para afrontar desafíos del mundo real. Tanto si eres principiante como si ya tienes conocimientos previos, este curso te ayudará a desarrollar las habilidades necesarias para destacar en el mundo de la programación.
